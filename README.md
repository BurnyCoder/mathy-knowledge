# Mathy Knowledge

An Obsidian vault for building a connected knowledge base around mathematics and mathematically grounded fields.

The vault is organized as short Markdown notes with Obsidian wikilinks, formulas, and source references. It is currently early-stage: some notes are developed, while others are placeholders for future expansion.

## Structure

- `Applied mathematics/` - notes connecting mathematics to AI, physics, neuroscience, computation, consciousness, information theory, and related topics.
- `Pure mathematics/` - foundations and core mathematical areas such as algebra, calculus, logic, set theory, category theory, linear algebra, and probability.
- `Words/` - vocabulary and conceptual notes that support the main knowledge graph.

## Current Focus

- AI and machine learning concepts
- Mathematical foundations and definitions
- Physics and computation topics
- Intelligence, consciousness, and related formal frameworks

## Using This Vault

Open the folder in Obsidian to browse the graph and follow `[[wikilinks]]` between notes. The Markdown files can also be read directly on GitHub, though Obsidian-specific links and backlinks work best inside Obsidian.

## Citation Style

Notes use inline references and footnote-style citations where relevant. Source quality and coverage will improve as the vault grows.
